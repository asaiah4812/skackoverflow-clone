About Stack Overflow Clone

This is a Stack Overflow clone built with Flask, a lightweight WSGI web application framework in Python.

Features

  • User authentication and profile management
  • Ask and answer questions
  • Vote on questions and answers
  • Tag questions for better organization
  • Search and filter questions
  • Markdown support for rich text formatting

Technologies Used

  • Flask - Web framework
  • SQLAlchemy - ORM for database operations
  • Flask-Login - User session management
  • Flask-WTF - Form handling and validation
  • Bootstrap 5 - Frontend framework
  • Jinja2 - Templating engine

This project was created as a learning exercise to demonstrate Flask application development with a focus on building a complex, interactive web application.
